# tt-random-js

Typing Tube にランダム機能 ( #random ) がありますが，動作しないんですよね．
ということで，JavaScript でランダム機能を実装してみました．
ブックマークレットにして使用してくださいね．

( ブックマークレットとは，ブラウザのブックマークに JavaScript のコードを登録して，クリックするだけで実行できる機能です． )

## 使い方

### 自分でコンパイル

1. [コード](./tt-random.js) をコピー
2. フリーのコンパイラで1行のコードに変換 ( javascript:abcdefghijklmn... のような形式 )
3. ブラウザのブックマークにurlとして追加 ( タイトルはお好みで )
4. Typing Tube でブックマークレットをクリック

### コンパイル済みを使用

1. [コンパイル済みコード](./tt-random_compiled.txt) をコピー
2. ブラウザのブックマークにurlとして追加 ( タイトルはお好みで )
3. Typing Tube でブックマークレットをクリック

### ランダム機能

Typing Tube 上でスクリプトを実行すると，ランダム検索画面が表示されます．
指定したレベルのデータがランダムに取得されます．

### 提案機能

対戦ページにてランダム検索機能を使用すると，「提案する」ボタンが表示されます．
クリックすると，ランダム選出したデータを **チャット** と **提案リスト** の両方に送信します．
